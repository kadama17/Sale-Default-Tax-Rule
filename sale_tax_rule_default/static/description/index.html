🚀 Global Tax for Quotations

One Click – All Lines Taxed!

Sale Default Tax Rule adds a powerful and time-saving feature to your Odoo sales workflow: apply a single tax to all lines of a quotation, instantly. No need to manually set taxes line by line – just choose one and let the module handle the rest.

🎯 Features

  • 🧾 Add a “Global Tax” field to sales quotations
  • 💡 Automatically apply selected tax to all order lines
  • ⚙️ Works with both tax-included and tax-excluded settings
  • 📦 Compatible with existing and new quotations
  • 🧩 Fully integrated with Odoo 17 (Community)

📥 Installation Guide

  1. Unzip the module inside your Odoo addons folder
  2. Restart your Odoo server
  3. Activate the Developer Mode
  4. Go to Apps → Update Apps List
  5. Search for Sale Default Tax Rule and click Install

🖼️ How It Works – Step by Step

Step 1: Select the Global Tax field on the quotation form

Step 2: All quotation lines are automatically updated
